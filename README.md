## 🚀 Instalando Projeto

Para instalar o projeto, siga estas etapas:

Clone o repositório
```
https://github.com/slashxu/nlw-copa.git
```

Baixe as dependências
```
npm install ou yarn install
```

Rodando o app
```
npx expo start
```
